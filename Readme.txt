Шахматы – это классическая настольная игра, которая была реализована как проект на языке Python. Для создания шахматной игры в Python использована графическую библиотеку Pygame для отображения игрового поля и фигур. В данном примере создается игровое поле шахматной доски и выводится в консоль. Была добавлена логика ходов фигур, проверка правил шахмат и реализация всей игры. Реальная игра шахмат включает в себя множество деталей, таких как логика движения каждой фигуры, проверка на шах и мат, возможность рокировки и взятия на проходе, в этом коде были реализованы данные механики
game.py Функции, входящие в файл необходимы для создания и работы интерфейса. Они отрисовывают доску, фигуры, доступные ходы, сообщения, выводимые пользователю, принимают и обрабатывают щелчки мышкой. 
piece.py Файл содержит классы, описывающие шахматные фигуры.
chess.py общий класс pygame.sprite.Sprite, содержащий код, необходимый для хранения координат фигур и их перерисовки. Каждый класс фигуры, возвращает список клеток, на которые могла бы походить фигура.
utils.py Проверка на произведенное нажатие мыши
res картинки для игры
